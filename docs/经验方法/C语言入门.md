## 为什么要会自行解决问题
- 大学之后很多知识都要自学而不是等别人教，自学能力很重要
- c语言考试不能得到助教的帮助，一味依赖助教可能导致考试时自己不会找代码的问题而难以取得满意的成绩

## 如何自行解决问题

- 上网查报错、语法等
- debug（视频以`devc++`为例）:

    - 了解常见评测结果的含义
    - 肉眼看代码
    - 单步调试
    - printf输出信息观察

- debug的数据问题

    - 手捏
    - 简单的自动生成
